PROJECT TITLE: Pentominoes
PURPOSE OF PROJECT: Implement an algorithm that finds a solution for a field with a certain sizes and a selection of pentomino blocks
VERSION or DATE: 12/10/2023
HOW TO START THIS PROJECT: Go to StartPentominoes.java and run the code
AUTHORS: Jonas, Mika, Allan, Vicente, Sasha, Laurens and Sofia
USER INSTRUCTIONS: after running it the code will ask for a horizontal and vertical size for the grid and the pentominoes you want to be used.
If the field cannot be created or the given characters are not actual pentominoes, the code will tell you this and end.
